# Node TypeScript API
API using Node TypeScript Jest and MongoDB

## Stack
- Dev
  - Node
  - TypeScript
  - Mongoose
  - MongoDB
- Test
  - Jest
  - Mockgoose

## Execute the API
```sh
$ npm install
$
$ npm start
```

## Test the API
```sh
$ npm run test
```
